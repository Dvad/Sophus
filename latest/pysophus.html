static vee(mat_omega)#

vee-operator It takes the 3x3-matrix representation mat_omega and maps it to the corresponding vector representation of Lie algebra.

This is the inverse of the hat-operator, see above.

Precondition: mat_omega must have the following structure:

0 -c b |
c 0 -a |
-b a 0 |
