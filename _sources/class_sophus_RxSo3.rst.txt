To be more specific, thixs function computes ``expmat(hat(omega))`` with ``expmat(.)`` being the matrix exponential and ``hat(.)`` being the :ref:`hat() <doxid-classsophus_1_1_rx_so3_1a5086cf43d17f15eabac637b7534f9eda>` -operator of RSO3.

Returns the ith infinitesimal generators of ``R+ x SO(3)``.

The infinitesimal generators of :ref:`RxSo3 <doxid-classsophus_1_1_rx_so3>` are:

.. ref-code-block:: cpp

	      |  0  0  0 |
	G_0 = |  0  0 -1 |
	      |  0  1  0 |
	
	      |  0  0  1 |
	G_1 = |  0  0  0 |
	      | -1  0  0 |
	
	      |  0 -1  0 |
	G_2 = |  1  0  0 |
	      |  0  0  0 |
	
	      |  1  0  0 |
	G_2 = |  0  1  0 |
	      |  0  0  1 |

Precondition: ``i`` must be 0, 1, 2 or 3.

hat-operator

It takes in the 4-vector representation ``a`` (= rotation vector plus logarithm of scale) and returns the corresponding matrix representation of Lie algebra element.

Formally, the :ref:`hat() <doxid-classsophus_1_1_rx_so3_1a5086cf43d17f15eabac637b7534f9eda>` -operator of :ref:`RxSo3 <doxid-classsophus_1_1_rx_so3>` is defined as

``hat(.): R^4 -> R^{3x3}, hat(a) = sum_i a_i * G_i`` (for i=0,1,2,3)

with ``G_i`` being the ith infinitesimal generator of :ref:`RxSo3 <doxid-classsophus_1_1_rx_so3>`.

The corresponding inverse is the :ref:`vee() <doxid-classsophus_1_1_rx_so3_1ac383036917ed13cbbc9372b77c887728>` -operator, see below.

Lie bracket.

It computes the Lie bracket of RxSO(3). To be more specific, it computes

``[omega_1, omega_2]_rxso3 := vee([hat(omega_1), hat(omega_2)])``

with ``[A,B] := AB-BA`` being the matrix commutator, ``hat(.)`` the :ref:`hat() <doxid-classsophus_1_1_rx_so3_1a5086cf43d17f15eabac637b7534f9eda>` -operator and ``vee(.)`` the :ref:`vee() <doxid-classsophus_1_1_rx_so3_1ac383036917ed13cbbc9372b77c887728>` -operator of :ref:`RxSo3 <doxid-classsophus_1_1_rx_so3>`.

vee-operator

It takes the 3x3-matrix representation ``Omega`` and maps it to the corresponding vector representation of Lie algebra.

This is the inverse of the :ref:`hat() <doxid-classsophus_1_1_rx_so3_1a5086cf43d17f15eabac637b7534f9eda>` -operator, see above.

Precondition: ``Omega`` must have the following structure:

.. code-block:: cpp

	|  d -c  b |
	|  c  d -a |
	| -b  a  d |
